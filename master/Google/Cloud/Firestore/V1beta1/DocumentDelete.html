class DocumentDelete extends Message

A [Document][google.firestore.v1beta1.Document] has been deleted.

May be the result of multiple [writes][google.firestore.v1beta1.Write], including updates, the last of which deleted the [Document][google.firestore.v1beta1.Document]. Multiple [DocumentDelete][google.firestore.v1beta1.DocumentDelete] messages may be returned for the same logical delete, if multiple targets are affected.
