DocumentChange

class DocumentChange extends Message

A [Document][google.firestore.v1beta1.Document] has changed.

May be the result of multiple [writes][google.firestore.v1beta1.Write], including deletes, that ultimately resulted in a new value for the [Document][google.firestore.v1beta1.Document]. Multiple [DocumentChange][google.firestore.v1beta1.DocumentChange] messages may be returned for the same logical change, if multiple targets are affected.
